# FirstKorea



### Version
----

  #### Swift Version
   * Swift 5
  #### Xcode Version
  * 12.4
  #### IOS Deployment Target
  * 14.4
 ------
### CocoaPods Library

* 'KakaoSDKCommon' - 필수 요소를 담은 공통 모듈
* 'KakaoSDKAuth' - 카카오 로그인
* 'KakaoSDKUser' - 사용자 관리
* 'Kanna' - 크롤링 모듈
* 'FSCalendar' - 달력 모듈
* 'Firebase/Core' - Firebase 모듈
* 'Firebase/Auth' - Firebase Auth 인증 모듈
* 'GoogleSignIn' - Google Auto 인증 모듈
